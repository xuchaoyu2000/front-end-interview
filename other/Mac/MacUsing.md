# Mac Using 
## Shortcut keys
- caps lock : 切换中英文
- cmd + w : 关闭当前页
- cmd + m : 最小化
- cmd + 1-9 : chrome下快速切换到相关标签
- cmd + l : 焦点移到地址栏
- cmd + t : 新增标签页
- cmd + r : 刷新
- cmd + shift + r : 强制刷新
- cmd + 空格 : 打开搜索栏
- 打开搜索栏后输入ter : 打开终端
- cmd + shift + n : 新建文件夹
- cmd + shift + g : 直达绝对路径文件夹
- cmd + shift + . : 显示／隐藏隐藏文件（夹）
- cmd + shift + del : 在finder下清空回收站
